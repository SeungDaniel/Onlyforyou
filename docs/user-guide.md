# 📖 사용자 가이드 (User Guide)

안녕하세요 Miin님! **건강 관리 서비스(펫 로봇)** 사용법을 알려드릴게요. 🐶

## 1. 시작하기
텔레그램 채팅방에서 **`/start`**를 입력하세요.
봇이 반갑게 인사하며 메인 메뉴를 보여줍니다.

## 2. 메인 메뉴
채팅방 하단이나 `/start` 입력 시 나타나는 버튼을 이용하세요.
- **[⚙️ 설정]**: 알림 시간을 내 마음대로 바꿀 수 있어요.
- **[🎵 음악]**: 기분에 딱 맞는 클래식 음악을 추천해드려요.
- **[ℹ️ 안내]**: 사용법을 다시 확인합니다.

## 3. 알림 시간 설정하기 (대화형 설정)
명령어를 외울 필요 없이, 친구랑 대화하듯 설정해보세요!

1. **[⚙️ 설정]** 버튼을 누르거나, 채팅창에 **"설정"**이라고 입력하세요.
2. 봇이 "어떤 시간을 바꾸시겠어요?"라고 물으면 버튼을 선택하세요. (예: **[기상 (비피더스)]**)
3. 원하는 시간을 편하게 말해주세요.
    - "오전 7시 30분"
    - "밤 10시반"
    - "14시"
4. 봇이 "네! 설정했어요!"라고 대답하면 완료!

## 4. 건강 루틴 알림
정해진 시간이 되면 봇이 알림을 보냅니다.
- **기상**: 비피더스 + 물 챙기기
- **오전**: 홍삼/로얄젤리 + 안타플러스
- **점심**: 아해티 + 하이파워 + 비타민C
- **저녁**: 랜덤 운동 미션 (스쿼트, 기지개 등)

알림을 받고 미션을 수행한 뒤 **[✅ 완료!]** 버튼을 꼭 눌러주세요. sh님께 보고가 올라갑니다! (가짜 보고지만 기분은 좋잖아요? 😉)

## 5. 음악 추천 받기
1. **[🎵 음악]** 버튼을 누르거나 `/music`을 입력하세요.
2. 지금 기분을 선택하세요.
    - **[☀️ 기분 좋아]**: 신나고 밝은 곡
    - **[☁️ 우울해]**: 위로가 되는 따뜻한 곡
    - **[🌙 쉬고 싶어]**: 편안한 휴식 곡
3. sh님이 엄선한 클래식 명곡(조성진, 임윤찬 등) 링크를 보내드립니다.
