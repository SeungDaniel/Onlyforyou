# Health Bot (건강 관리 펫 로봇)

## 📖 개요
**Health Bot**은 바쁜 일상 속에서 건강을 챙기기 힘든 사용자를 위해 제작된 텔레그램 봇입니다.
남자친구(sh)가 보낸 **'반려 로봇'** 페르소나를 가지고 있으며, 다정하고 귀여운 말투로 영양제 섭취, 운동, 휴식을 챙겨줍니다.

## ✨ 주요 기능
1.  **건강 루틴 알림**: 기상, 오전, 점심, 저녁(운동) 시간에 맞춰 맞춤형 알림을 보냅니다.
2.  **대화형 스케줄 설정**: `/setup` 명령어나 "설정" 대화를 통해 자연어로 알림 시간을 변경할 수 있습니다.
3.  **상황별 음악 추천**: 기분(좋음/우울/휴식)에 따라 엄선된 클래식 음악(조성진, 임윤찬 등)을 추천합니다.
4.  **랜덤 미션**: 매번 다른 알림 멘트와 랜덤 운동 미션(스쿼트, 기지개 등)을 제공합니다.
5.  **데이터 저장**: 사용자 설정(알림 시간)은 `user_data.json`에 자동 저장되어 봇이 재시작되어도 유지됩니다.

## 🚀 시작하기
1.  `requirements.txt` 설치: `pip install -r requirements.txt`
2.  `.env` 파일 설정: `TELEGRAM_TOKEN` 입력
3.  봇 실행: `python main.py`
4.  텔레그램에서 `/start` 입력
